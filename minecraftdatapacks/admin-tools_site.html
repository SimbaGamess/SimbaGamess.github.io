Admin Tools

With this Datapack you get new Admin commands and Tools.
Commands:

/function admin:admin_on
with this command you will be set in Admin mode,
the player mode changes to creative,
a prefix is addet to the name [Admin]
aswell your current survival Inventory will be saved,
to load it back later.
U will get an Admin_Sword and a nice Looking Helmet with Horns
(only works with Texturpack)

/function admin:admin_off
with this command you will be set in Normal mode,
the player mode changes to survival,
the prefix is removed
your safed inventory now loads back.

/function admin:heal
